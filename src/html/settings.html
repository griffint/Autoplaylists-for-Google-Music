Settings

The number of minutes to wait between syncing your autoplaylists with Google.
Lower numbers sync more often, keeping playlists more accurate at the cost of more ui refreshes and higher mobile battery use.
Regardless of this setting, changing an autoplaylist definition will always sync that playlist immediately.
Upon changing this setting, a sync will occur immediately (unless set to 0).

Check to opt in to the new sync apis. These should improve performance and sync accuracy, but are still experimental.
